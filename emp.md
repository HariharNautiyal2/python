# Program that calculates GST And Insurance rate
Python program that calculates GST (30%) and insurance (2%) of an employee's salary:

<br>

```python
salary = int(input("Tell the salary: "))
gst = 30/100 * salary
etc = 2/100 * salary

print("Salary: ",salary)
print("Gst: ",gst)
print("ETC: ",etc)
print("Remaning slry ",salary - etc - gst)

```
